# Quiz App

An interactive quiz application built with **JavaScript**, **HTML**, and **CSS**.  
The app presents multiple-choice questions, lets users select answers, and calculates the final score.

---

## 🚀 Features
- Clean and user-friendly interface.
- Dynamic question rendering from an array or data file.
- Automatic score calculation after completing the quiz.
- Option to restart the quiz.
- Responsive design that works on desktop and mobile.
